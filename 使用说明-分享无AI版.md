# Job Application Helper 使用说明（无需安装 AI）

## 1. 安装扩展

1. 右键收到的 ZIP，选择 **全部解压缩**。
2. 打开 Chrome，进入 `chrome://extensions`。
3. 打开右上角 **开发者模式**。
4. 点击 **加载已解压的扩展程序**。
5. 选择解压后的 `JobApplicationHelper` 文件夹。

## 2. 导入已经处理好的资料

1. 点击扩展图标。
2. 点击 **Open profile manager**。
3. 展开 **Import or export a processed profile**。
4. 点击 **Import processed profile**。
5. 选择收到的 JSON 资料文件。
6. 检查资料后点击 **Save profile**。

## 3. 填写申请表

1. 打开职位申请页面。
2. 点击扩展图标。
3. 点击 **Scan this page**。
4. 检查扫描结果。
5. 对确定的内容点击 **Confirm and fill** 或每行的 **Fill this**。
6. 对漏掉的格子，点击 **Enable click-to-choose helper**，再点击网页中的格子，从旁边浮窗选择资料。

## 4. 注意

- 这个版本不需要安装 Ollama，也不需要下载 AI model。
- 扩展不会自动点击提交按钮。
- 上传简历文件仍然需要手动选择，浏览器不允许扩展自动选择本地文件。
- JSON 资料包含个人信息，请只通过可信渠道分享。
