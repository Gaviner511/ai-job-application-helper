# Job Application Helper 使用说明（完整版）

## 1. 准备 Profile

1. 打开扩展，点击 **Open profile manager**。
2. 手动填写资料，或上传简历后点击 **Classify with local AI**。
3. 检查每张 Education、Work Experience、Project、Certification、Language 卡片。
4. 点击 **Save profile**。

## 2. 本地 AI

本地 AI 只在你自己的电脑上使用 Ollama。你需要先启动 Ollama，并在 Profile Manager 里点击 **Detect models** 和 **Test local AI**。

如果只是填写已经保存好的 Profile，不一定要用 AI。

## 3. 填写申请表

1. 打开职位申请页面。
2. 点击扩展图标。
3. 点击 **Scan this page**。
4. 检查结果。
5. 可以点击 **Confirm and fill** 批量填写，也可以点每一行的 **Fill this**。
6. 如果有漏掉的格子，点击 **Enable click-to-choose helper**，再点击网页里的输入框，从旁边浮窗选择资料。

## 4. 测试和重填

- **Fill blank fields only** 默认开启，不会覆盖网页上已有内容。
- **Clear visible fields on this page** 可以清空当前页面可见可编辑字段，方便测试。
- 扩展不会自动点击最终提交按钮。
