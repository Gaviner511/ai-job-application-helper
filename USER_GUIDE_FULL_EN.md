# Job Application Helper User Guide (Full Version)

## 1. Set Up Your Profile

1. Open the extension and click **Open profile manager**.
2. Fill your profile manually, or upload a resume and click **Classify with local AI**.
3. Review each Education, Work Experience, Project, Certification, and Language card.
4. Click **Save profile**.

## 2. Local AI

Local AI uses Ollama on your own computer. Start Ollama first, then use **Detect models** and **Test local AI** in Profile Manager.

AI is optional for filling saved profile data, but useful for reading resumes and improving page scans.

## 3. Fill an Application

1. Open a job application page.
2. Click the extension icon.
3. Click **Scan this page**.
4. Review the matched fields.
5. Use **Confirm and fill** for batch filling, or **Fill this** for one field.
6. For missed fields, enable **click-to-choose helper**, click a field on the page, and choose a value from the floating panel.

## 4. Testing

- **Fill blank fields only** is enabled by default.
- **Clear visible fields on this page** clears editable fields for testing.
- The extension never clicks the final submit button.
