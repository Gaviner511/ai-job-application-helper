# Job Application Helper User Guide (No-AI Version)

## 1. Install the Extension

1. Right-click the ZIP file and choose **Extract all**.
2. Open Chrome and go to `chrome://extensions`.
3. Turn on **Developer mode**.
4. Click **Load unpacked**.
5. Select the extracted `JobApplicationHelper` folder.

## 2. Import a Processed Profile

1. Click the extension icon.
2. Click **Open profile manager**.
3. Expand **Import or export a processed profile**.
4. Click **Import processed profile**.
5. Select the JSON profile file you received.
6. Review the profile and click **Save profile**.

## 3. Fill an Application

1. Open a job application page.
2. Click the extension icon.
3. Click **Scan this page**.
4. Review the results.
5. Use **Confirm and fill** or **Fill this**.
6. For missed fields, enable **click-to-choose helper**, click a field on the page, and choose the right profile value.

## 4. Notes

- You do not need Ollama or an AI model.
- The extension never clicks the final submit button.
- Resume file uploads still need to be selected manually.
- The JSON profile contains personal information. Share it only through trusted channels.
